# 🎓 Student Marks Manager (Python)

A command-line Student Marks Manager built using Python. This project allows users to manage student records by adding, searching, updating, deleting, and displaying student information through a simple menu-driven interface.

## 🚀 Features

- ✅ Add Student
- ✅ Prevent Duplicate Student Names
- ✅ Search Student
- ✅ Update Student Marks
- ✅ Delete Student
- ✅ Display All Students
- ✅ Menu-Driven Interface

## 🛠 Technologies Used

- Python 3

## 📚 Python Concepts Used

- Functions
- Lists
- Nested Lists
- Loops (`for`, `while`)
- Conditional Statements (`if`, `elif`, `else`)
- Boolean Flags
- `break` and `return`
- User Input
- Menu-Driven Programming
